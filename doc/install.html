Install  
Instructions for installing pyftpdlib (for those really new to Python).

Requirements¶

Python 2.3 or higher (not Python 3.x). You can get the latest 2.x stable Python release from here: http://www.python.org/download/.

Using easy_install / setuptools¶

If you have easy_install / setuptools on your system, installing pyftpdlib is quite simple. Just run: 

easy_install pyftpdlib

This will get the most updated pyftpdlib from the Python pypi repository, unpack it and install it automatically.

Note: if you already have an old version of pyftpdlib installed, easy_install will not automatically download the latest version. You can ask for a particular version by running, for example: 

easy_install.py pyftpdlib==0.3.0

Manual installation¶

If you have downloaded a pyftpdlib package, follow the following steps:

Unpack it (Windows users could use 7Zip, WinRar or other similar program): 

tar zxvf pyftpdlib-0.3.0.tar.gz

Change to the pyftpdlib directory: 

cd pyftpdlib

Run setup.py to install pyftpdlib. This step need to be run as root. 

python setup.py install

If you're on Windows just run: 

setup.py install
